###### 【程式開發練習1】溫度轉換

```
華氏溫度轉換成攝氏溫度

華氏 = 攝氏*(9/5)+32

攝氏 = (華氏-32)*5/9
```
====================================================
# C
```
#include<stdlib.h> 
#include<stdio.h> 
int main() 
{ 
    int a; 
    printf("輸入華氏溫度"); 
    scanf("%d",&a); 
    printf("攝氏溫度為%d\n",(a-32)*5/9); 
    
    return 0; 
}
```
